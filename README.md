## ✨ Main Features

> ### 🎯 **실시간 자세 분석**
> MediaPipe를 활용한 33개 관절 실시간 추적 및 정확도 분석

> ### 🤖 **AI 기반 체형 분석** 
> 개인별 체형 데이터를 분석하여 맞춤형 운동 가이드라인 생성

> ### 📊 **3D 좌표계 변환**
> 사람 중심 좌표계로 변환하여 정확한 운동 자세 교정

> ### 🚨 **낙상 감지 시스템**
> TensorFlow 기반 낙상 감지 및 Twilio 응급 연락 서비스

> ### 🎵 **음성 가이드**
> 운동 횟수 카운팅 및 실시간 음성 피드백 제공

> ### 📱 **실시간 통신**
> Socket.io를 통한 실시간 데이터 송수신 및 저지연 처리

## 🚀 Git Flow

```mermaid
gitgraph
    commit id: "Initial commit"
    branch develop
    checkout develop
    commit id: "Setup project"
    branch feat/pose-detection
    checkout feat/pose-detection
    commit id: "Add MediaPipe"
    commit id: "Implement pose analysis"
    checkout develop
    merge feat/pose-detection
    branch feat/ai-model
    checkout feat/ai-model
    commit id: "Add TensorFlow"
    commit id: "Train fall detection"
    checkout develop
    merge feat/ai-model
    checkout main
    merge develop
```

- 🌟 **`main`** - 프로젝트 최종 merge, 배포 가능한 브랜치
- 🚀 **`develop`** - 완성한 기능들을 merge, 항상 배포 가능한 상태 유지  
- ⚡ **`{type}/{description}`** - 개발 브랜치 (예: `feat/login`, `fix/login-token`)

## 🏗️ Project Structure

```
PoseSync/
├── 🐍 Backend (Flask)
│   ├── app/
│   │   ├── ai/                 # AI 모델 & 체형 분석
│   │   ├── controllers/        # API 컨트롤러
│   │   ├── models/            # 데이터베이스 모델
│   │   ├── services/          # 비즈니스 로직
│   │   ├── sockets/           # Socket.io 이벤트
│   │   └── util/              # 유틸리티 함수
│   └── requirements.txt
│
├── ⚛️ Frontend (React + TypeScript)
│   ├── src/
│   │   ├── components/        # 재사용 컴포넌트
│   │   ├── hooks/            # Custom Hooks
│   │   ├── pages/            # 페이지 컴포넌트
│   │   ├── types/            # TypeScript 타입
│   │   └── utils/            # 유틸리티 함수
│   └── package.json
│
└── 📊 Database
    ├── ERD/                  # 데이터베이스 설계
    └── migrations/           # DB 마이그레이션 파일
```

## 🚀 Quick Start

### 1️⃣ Clone Repository
```bash
git clone https://github.com/PoseSync/PoseSync.git
cd PoseSync
```

### 2️⃣ Backend Setup
```bash
cd backend
pip install -r requirements.txt
python app.py
```

### 3️⃣ Frontend Setup  
```bash
cd frontend
npm install
npm start
```

### 4️⃣ Environment Variables
```bash
# .env 파일 생성
DATABASE_URL=mysql://username:password@localhost/posesync
TWILIO_ACCOUNT_SID=your_twilio_sid
TWILIO_AUTH_TOKEN=your_twilio_token
AWS_ACCESS_KEY_ID=your_aws_key
AWS_SECRET_ACCESS_KEY=your_aws_secret
```
